I might be late to the game but I’ve been developing a personal iOS app I have wanted for years, but never had the time or chance to dedicate a few weeks to. In a few hours, I have a functional app and I only need to tweak a few things here and there. The code is also good and well structured but has zero tests (but I didn’t ask it to choose TDD per se).

I only used Claude Code (from the app, not terminal) in Plan + Build mode without any tools or plugins. This is wild.

On top of it, I asked Claude to use `fastlane` to setup my my Apple Developer account so I could install it on my phone.

I’m writing this post from that application. Such a wild time.
